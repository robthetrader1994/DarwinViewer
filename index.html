Darwin Stable

Lines starting with # are folder names; darwins go under them. Edit freely and Save. This is the quickest way to move darwins between folders.

  1. Open the darwin's page on darwinexzero.com.
  2. DevTools: Ctrl+Shift+I → Network → Fetch/XHR.
  3. Filter box: type productquotes, then refresh the page.
  4. Read the full name off a row, e.g. XXXX.5.2 (never guess the suffix).
  5. Click + on the folder you want, paste the name, Add.
